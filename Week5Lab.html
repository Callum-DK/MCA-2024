Which MEI Documentation Elements Correspond?

Used pubStmt and publisher to adhere to metadata standards, used “corpName” from textual content from the metadata standards webpage for MEI.

titleStmt Container for title and responsibility meta-data. Used title element from textual content to link the wiki page for guardians of the galaxy (original score) for the album, description, and genre where my piece is a part of including its own title.

respStmt (responsibility statement) – Transcription of text that names one or more individuals, groups, or in rare cases, mechanical processes, responsible for creation, realization, production, funding, or distribution of the intellectual or artistic content.

Used persName as it is related and part of the textual content section of respStmt. This was used when creating data link for the composer with the relevant authority.

(personal name) – Designation for an individual, including any or all of that individual's forenames, surnames, honorific titles, and added names.

Using parent element Publisher and textual content or child element corpName I was able to implement the publisher as Marvel Music Incorporated.

Using element distributor I was able to refer to the metadata standards and find textual content child element repository in order to data link the notation file and sheet notation application musescore as the relevant authority.

distributor Person or agency, other than a publisher, from which access (including electronic access) to a bibliographic entity may be obtained.

repository Institution, agency, or individual which holds a bibliographic item.
